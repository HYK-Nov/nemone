## 📁프로젝트 구조
```
Nemone/
├── Controller/           # 애플리케이션의 제어 로직
├── Form/                 # 사용자 인터페이스 폼
├── Model/                # 데이터 모델 및 비즈니스 로직
├── Service/              # 서비스 계층 로직
├── Resources/            # 리소스 파일 (이미지, 문자열 등)
├── Properties/           # 프로젝트 속성 및 설정
├── MainHubForm.cs        # 메인 폼 코드
├── Program.cs            # 애플리케이션 진입점
├── App.config            # 애플리케이션 구성 파일
├── Nemone.sln            # 솔루션 파일
└── README.md             # 프로젝트 설명서
```
